# iBut考试对策学习网页 Spec v1.0

## 1. 项目简介

本项目是一个用于复习 iBut 考试对策资料的静态学习网页。

目标是把原本文档中的章节知识、关键词中日英对照、考试题眼、易混淆点和模拟题，整理成一个方便逐章查看、快速检索、移动端可阅读的学习工具。

本项目只做静态前端页面，不做后端，不做登录，不做数据库。

---

## 2. 项目开发宪法

本项目必须严格遵守以下规则：

1. 只使用 HTML + CSS + 原生 JavaScript
2. 不使用任何前端框架
   - 禁止 React
   - 禁止 Vue
   - 禁止 Next.js
   - 禁止 Tailwind
   - 禁止 Bootstrap
3. 布局优先使用 FCC 已学过的 Flex 响应式布局
4. 必须使用语义化 HTML 标签
5. 代码缩进统一，格式清晰
6. 所有 AI 生成代码必须能逐行读懂
7. 禁止复制自己看不懂的代码
8. 禁止使用 `_` 忽略逻辑
9. JS 只写简单、可解释、可修改的逻辑
10. 每个主要代码块必须有简单中文注释
11. 先写 spec.md，再写代码
12. 所有页面必须适配移动端
13. 代码变更必须通过 Git 规范提交

---

## 3. 页面整体结构

页面包含以下主要模块：

1. 顶部标题区 Header
2. 章节导航区 Nav
3. 搜索与筛选区 Search / Filter
4. 左侧章节目录区 Sidebar
5. 右侧内容阅读区 Main Content
6. 当前章节关键词中日英对照区 Terms Table
7. 题眼速查区 Exam Hints
8. 模拟题练习区 Quiz
9. 页脚 Footer

---

## 4. 顶部标题区 Header

Header 必须包含：

- 项目标题：iBut考试对策学习宫殿
- 简短说明：
  - 逐章学习
  - 关键词中日英对照
  - 考试题眼速查
  - 模拟题复习
- 当前学习进度：
  - 已读板块数 / 总板块数

示例：

```text
iBut考试对策学习宫殿
按章节学习 · 按题眼速查 · 按关键词复习
已读 3 / 20
